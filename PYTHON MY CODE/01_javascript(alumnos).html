Bienvenidos a la Introducción a JavaScript

En esta clase exploraremos los conceptos clave de JavaScript, el lenguaje de programación que añade interactividad a la web.

Variables en JavaScript

Las variables son contenedores que almacenan datos. Puedes crear variables con let, var o const.

MI CODE

Condiciones en JavaScript

Las condiciones permiten tomar decisiones dentro del código.

Bucles en JavaScript

Los bucles permiten ejecutar un bloque de código repetidas veces.

    Funciones en JavaScript

    Las funciones son bloques de código reutilizables.

    Eventos en JavaScript

    Los eventos son interacciones que el usuario tiene con la página.

    Arrays en JavaScript

    Un array es una colección de datos almacenados en posiciones llamadas índices.

    Objetos en JavaScript

    Un objeto es una estructura que agrupa pares clave-valor.
